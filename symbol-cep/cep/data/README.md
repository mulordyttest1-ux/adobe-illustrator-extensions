# Symbol CEP — Presets Storage

This folder contains user configuration presets for the Symbol CEP panel.

## presets.json
Generated automatically by the panel on first use.  
**This file is intentionally committed to Git** so that presets are portable across machines.

## How to share presets
1. Commit this repository (or copy the `symbol-cep/cep/data/presets.json` file).
2. On the new machine, install the extension and open Illustrator.
3. The presets will appear automatically in the Actions tab.
